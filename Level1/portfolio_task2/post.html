Image Resizer

This was a project I started in February 2022 when I first learned Electron JS.This project was made keeping in mind that there is some traction towards web-apps in the market.

The system takes images from your system and then according to your needs adjusts the size of the image.It is a web app made with the help of web-app Framework Electron JS which also integrates with React

The image when resized downloads in your local system. Many other features can also be to the same such as image extension converter etc.

I worked on this system for 2 months to gain expertise in my basic understanding of web technologies and also experience of ElectronJS

Technologies:
  • - JavaScript
  • - Express JS
  • - React JS
  • - Electron JS
